# SMO
Моделирование заправочной станции с 1 местом в очереди.
В системе имеются:
  - пул заявок,
  - пул обслуживания (x2),
  - пул очереди на обслуживание,
  - пул обслуженных заявок,
  - пул необслуженных заявок.
При желании систему можно доработать на N мест в очереди алгоритмически.
